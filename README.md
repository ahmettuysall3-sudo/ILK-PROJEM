# ILK-PROJEM
PIC16F628A Assembly ile LED kontrol örneği.  PORTB üzerinden binary 9 değeri gösterilir ve delay için sayaç tabanlı döngü kullanılmıştır.






Bu projeye ait Proteus simülasyon dosyası `proteus` klasöründe bulunmaktadır.

Devre özellikleri:
- PIC16F628A kullanılmıştır
- PORTB üzerinden LED kontrolü yapılmaktadır
- Binary 9 (00001001) değeri gösterilmektedir

Simülasyonu çalıştırmak için:
1. Proteus projesini açın (.pdsprj)
2. Simülasyonu başlatın
